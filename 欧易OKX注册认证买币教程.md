# **充币提币怎么操作？**

当你需要在不同交易所之间转移加密货币，或者和别人转账USDT/比特币时，就需要用到充币提币功能。

**充币（收钱）：**

  1. 点击「资产」→「充币」
  2. 选择要充值的币种
  3. 选择网络（推荐TRC20，手续费便宜）
  4. 复制充值地址给对方
  5. 等待到账（通常几分钟）

**提币（转钱）：**

  1. 点击「资产」→「提币」
  2. 选择要提取的币种
  3. 输入对方的钱包地址
  4. 选择网络（必须和对方一致）
  5. 输入提币数量
  6. 确认提币，等待处理

记住，提币时网络选择必须和接收方一致，否则币会丢失。第一次转账建议先转小额测试。
